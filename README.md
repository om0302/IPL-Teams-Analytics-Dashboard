# 🏏 IPL Teams Analytics Dashboard

An interactive **Data Analysis and Visualization** project developed
using **Microsoft Power BI** and **Microsoft Excel** to analyze IPL
cricket data. The project transforms raw match data into meaningful
insights through data preprocessing, feature engineering, and
interactive dashboards.

------------------------------------------------------------------------

## 📌 Project Overview

This project analyzes IPL cricket match data to understand team
performance, scoring trends, wicket distribution, and run-rate patterns.
The analysis is presented through interactive Power BI dashboards that
help users explore batting and bowling performance using filters and
visualizations.

------------------------------------------------------------------------

## 🎯 Objectives

-   Analyze IPL batting and bowling performance.
-   Perform data preprocessing and feature engineering.
-   Build interactive Power BI dashboards.
-   Extract meaningful insights from historical IPL data.

## 📊 Dashboard Features

### 🏏 Batting Analysis

-   Total Runs
-   Average Score
-   Maximum Score
-   Runs by Team
-   Runs vs Overs
-   Average Run Rate
-   Team Contribution
-   Interactive Team Filter

### 🎯 Bowling Analysis

-   Total Wickets
-   Average Wickets
-   Wickets in Last 5 Overs
-   Wickets by Team
-   Wickets vs Overs
-   Wicket Contribution
-   Interactive Team Filter

------------------------------------------------------------------------

## 📈 Key Insights

-   Mumbai Indians recorded the highest total runs.
-   Chennai Super Kings showed consistent batting performance.
-   Delhi Daredevils and Kolkata Knight Riders recorded the highest
    wicket counts.
-   Run rate is a strong indicator of batting efficiency.
-   The final five overs have a significant impact on both runs and
    wickets.
